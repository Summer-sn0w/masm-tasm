# Notes

## 1

VSCode类型与汇编类型的对应

| assembly symbol | vscode symbol | 汇编关键字 | vscode关键字 |
| --------------- | ------------- | ---------- | ------------ |
| macro           | Module        | 宏         | 模块         |
| segment         | Class         | 段         | 类           |
| procedure       | Function      | 子程序     | 函数         |
| struct          | Struct        | 结构体     | 结构体       |
| label           | Key           | 标号       | 键           |
| variable        | Variable      | 变量       | 变量         |
